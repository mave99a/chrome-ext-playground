About

Demo Side Panel Extension

This is a demonstration Chrome extension showcasing the Side Panel API introduced in Chrome 114.

Features

  • Side panel interface with tabbed navigation
  • Quick access to external websites
  • Internal page viewer
  • Current tab metadata extraction

Technical Details

  • Manifest Version: 3
  • APIs Used: sidePanel, tabs, scripting
  • No external dependencies

Version

v1.0.0
